PredSolver (better name sought...)

Concepts

Concept

Purpose

Example & Notes

Var

A typed and named placeholder for values

X : Integer

S: String

D: Timestamp

Binding

The current state of a Variable, being either free (no values), or associated to zero, one, many values.

Ideally, Bindings may be able to store a finite or infinite set of values, exposing either data, functions, or iterators to access them.

Predicate

An association of several Bindings or Predicates in a way that can be proven true or false


Solver

The algorithm that can prove predicates


Tuple

A singleton, pair, triple, quad, etc or related solutions to a predicate.




Notes : we used to have a « Provider » that creates and owns predicates. This makes some sense because predicates are defined in a domain (the space to solve a problem), and need connections to data sources. They are definitely not floating in the air.
